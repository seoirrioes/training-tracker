# training-tracker

Kleine persönliche Trainings-WebApp für Svens 4-Tage-Rotation.

## Funktionen
- Tag 1 bis Tag 4 Rotation
- Trainingsplan hinterlegt
- Satz-für-Satz Eingabe von Gewicht und Wiederholungen
- lokaler Verlauf pro Übung
- Pausentimer
- Notizen pro Einheit

## Hosting
Die App ist als statische Seite ausgelegt und kann z. B. über GitHub Pages veröffentlicht werden.

## Hinweis zur Datenspeicherung
Aktuell werden Trainingsdaten lokal im Browser via `localStorage` gespeichert.
